## 🚀 Featured Projects

### 🧠 [Attendance Track — Agentic Data Intelligence Engine](https://github.com/bhattsarthak125/attendance-track)
An offline-first, agentic AI data extraction suite that runs entirely on local Linux infrastructure — no cloud inference required.
- Asynchronous multi-processing engine ingesting and parsing **10,000+ objects** with a minimal CPU thread footprint
- Local inference via **Ollama (llama3.1:8b)** to extract and convert unstructured input into validated JSON schemas
- Custom RegEx pre-filtering layer to strip irrelevant metadata and cut inference compute overhead
- Hybrid database layer: **SQLite** for deterministic relational lookups + **ChromaDB** for high-velocity vector search

`Python` `Linux` `Ollama` `SQLite` `ChromaDB` `Asyncio`

### 💬 [Terminal Whisper — Secure Shell-Native Messaging Platform](https://github.com/bhattsarthak125/terminal-whisper)
An open-source, terminal-native messaging utility built for remote-access communication workflows.
- Streams secure, low-latency command-line interactions over **Socket.IO**
- Enables concurrent multi-user messaging entirely inside raw shell terminal sessions
- Tested across multiple Linux distributions

`Node.js` `Socket.IO` `Terminal API` `Linux`
